Willkommen

Diese Seite wird Dir einen einzigartigen Code zuweisen. Je nachdem, an welchen anderen Studien unserer Arbeitsgruppe Du noch teilnimmst, verwenden wir den Code oder Deine E-Mail Adresse um Dich zu identifizieren. Damit Du Dich nicht mit dem langen Code beschäftigen musst, speichern wir den Code mit Hilfe eines Cookies auf Deinem Smartphone/iPhone.

Durch diese Form der Verschlüsselung können wir Deine persönlichen Daten vertraulich behandeln.

Auf der nächsten Seite wirst Du gebeten persönliche Daten wie z.B. Deinen Namen anzugeben. Bis auf die Eingabe Deiner E-Mail Adresse werden diese Eingaben jedoch NICHT gespeichert. Sie dienen ausschliessliche dem Erstellen des Codes. Es ist dennoch wichtig,dass Du wahrheitsgetreu antwortest, denn auch bei anderen Experimenten werden wir den Code erneut erstellen. So können wir Deine Angaben hier mit Deinen Angaben in anderen Studien unserer Arbeitsgruppe in Verbindung bringen. Wichtiger Hinweis: Es ist möglich mit Deinen persönlichen Angaben den Code zu erstellen. Andersherum können wir aus dem Code aber NICHT Deine persönlichen Angaben (z.B. Deinen Namen) herleiten. So schützen wir Deine Privatsphäre.

Indem Du auf „weiter“ klickst, akzeptierst Du, diese Bedingungen.

Eingaben zum Erstellen des Identifiers

Bitte merke Dir, mit welcher E-Mail Adresse Du Dich hier anmeldest. Am besten notierst Du Dir, welche E-Mail Adresse Du für i-BEAT verwenden möchtest, z.B. mit Deiner Notizen-App auf dem Smartphone oder iPhone. Fülle das Formular wahrheitsgetreu aus.
